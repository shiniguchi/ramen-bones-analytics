```bash
# 2. Set description
gh repo edit --description "Free, forkable, mobile-first analytics for restaurant owners. Turns Orderbird POS transactions into banking-grade cohort/retention/LTV metrics with nightly Claude-generated insights."

# 3. Set topics (one --add-topic per topic; gh supports multiple)
gh repo edit \
  --add-topic analytics \
  --add-topic restaurant-analytics \
  --add-topic sveltekit \
  --add-topic svelte \
  --add-topic supabase \
  --add-topic cloudflare-pages \
  --add-topic forkable \
  --add-topic pos-integration \
  --add-topic cohort-analysis

# 4. Verify
gh repo view --json visibility,description,repositoryTopics
```
