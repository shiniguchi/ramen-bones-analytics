## Critical Finding (read first)

**D-01 says "mirror `RevenueForecastCard.svelte`'s past/future split."** That split **does not yet exist** in the codebase. `RevenueForecastCard.svelte:148-159` and `InvoiceCountForecastCard.svelte:148-159` currently render forecast as a **single Spline+Area block per model**, undifferentiated past vs future. `CalendarRevenueCard.svelte:313-337` and `CalendarCountsCard.svelte` (mirror) do the same.

**What D-01 actually means:** introduce the split-shape on the Calendar* cards (the visible-feedback gap the friend-owner reported), and the Forecast* cards become consistent siblings later (deferred — out of scope per CONTEXT.md). The "source-of-truth" RevenueForecastCard reference describes the **per-model render shape** (Area-band-then-Spline-line ordering, naive_dow dashing, palette) that the new split must preserve, not a pre-existing split implementation.

**Partition primitive (planner must invent):** `last_actual_date = max(actuals[].date)`. The existing `actuals` array already lands at `forecastData.actuals` from `/api/forecast` (`+server.ts:171-174`). For each model's row array `modelRows`, partition by `r.target_date < last_actual_date` (= past) vs `r.target_date >= last_actual_date` (= future). CI band stays a **single continuous Area** spanning all rows (D-04); only the Splines get split into two.

**Existing fetch / state pattern (preserve verbatim, no change)** — `CalendarRevenueCard.svelte:69-108`:

```ts
let forecastData = $state<ForecastPayload | null>(null);
let visibleModels = $state(new Set<string>(['sarimax', 'naive_dow']));
let lastFetchedGrain = $state<string | null>(null);

$effect(() => {
  const grain = getFilters().grain as 'day' | 'week' | 'month';
  if (lastFetchedGrain === grain) return;
  lastFetchedGrain = grain;
  const url = `/api/forecast?kpi=revenue_eur&granularity=${grain}`;
  clientFetch<ForecastPayload>(url)
    .then((data) => { forecastData = data; })
    .catch(() => { forecastData = null; });
});

const seriesByModel = $derived.by(() => {
  const map = new Map<string, ForecastRow[]>();
  const rows = forecastData?.rows ?? [];
  for (const r of rows) {
    if (!visibleModels.has(r.model_name)) continue;
    if (!map.has(r.model_name)) map.set(r.model_name, []);
    map.get(r.model_name)!.push(r);
  }
  for (const arr of map.values()) {
    arr.sort((a, b) => a.target_date.localeCompare(b.target_date));
  }
  return map;
});
```

function formatEur(v: number | null | undefined): string {
  if (v === null || v === undefined) return '—';
  const sign = v >= 0 ? '+' : '−';
  return `${sign}€${Math.abs(Math.round(v)).toLocaleString('de-DE')}`;
}

**Honest-label / divergence derivations (preserve verbatim)** — `CampaignUpliftCard.svelte:94-112`:

```ts
const ciOverlapsZero = $derived.by(() => {
  if (!headline) return false;
  const lo = headline.row.ci_lower_eur;
  const hi = headline.row.ci_upper_eur;
  return lo <= 0 && hi >= 0;
});

const divergenceWarning = $derived.by(() => {
  if (!headline) return false;
  const n = headline.row.naive_dow_uplift_eur;
  if (n === null) return false;
  const s = headline.row.cumulative_uplift_eur;
  const signDisagree = Math.sign(s) !== Math.sign(n) && (s !== 0 || n !== 0);
  const magnitudeDivergent = Math.abs(s - n) / Math.max(Math.abs(s), 1) > 0.5;
  return signDisagree || magnitudeDivergent;
});
```
