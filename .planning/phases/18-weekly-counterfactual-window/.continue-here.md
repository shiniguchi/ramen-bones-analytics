## Mock Injection Technique (for Task 3 localhost QA)

Since `weekly_history` is empty in the real DB, use this pattern in Playwright `browser_evaluate`:

```javascript
// 1. Set fetch mock
window.fetch = async (url, opts) => {
  if (url.includes('/api/campaign-uplift')) {
    return new Response(JSON.stringify(MOCK_PAYLOAD), {status: 200, headers: {'Content-Type': 'application/json'}});
  }
  return origFetch(url, opts);
};

// 2. Clear clientFetch cache
const { invalidateClientCache } = await import('/src/lib/clientFetch.ts');
invalidateClientCache('/api/campaign-uplift');

// 3. Mount fresh component
const div = document.getElementById('qa-test') || (() => { const d = document.createElement('div'); d.id='qa-test'; document.body.appendChild(d); return d; })();
div.innerHTML = '';
const mod = await import('/src/lib/components/CampaignUpliftCard.svelte');
const { mount } = await import('/@fs/.../node_modules/svelte/src/index-client.js');
mount(mod.default, { target: div });
await new Promise(r => setTimeout(r, 1000));
```

Use `MOCK_PAYLOAD` with 5 weeks of `weekly_history` (sarimax model) and a `campaigns` block.

For **ja locale**: navigate to `http://localhost:5173/?lang=ja` before mounting, OR toggle locale via the UI switcher after mounting.
