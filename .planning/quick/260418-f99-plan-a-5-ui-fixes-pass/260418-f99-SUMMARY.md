## Fix 1 — padding path taken

**Top-level `<BarChart padding={{...}}>`**, not `props.padding`. Reason: BarChart's `props` slot is a keyed dispatch (`xAxis`/`yAxis`/`bars`/`tooltip`/`brush`) and does not forward arbitrary keys through to the underlying `<Chart>`. The top-level `padding` prop mirrors how `CohortRetentionCard` already calls it on `<Chart>` directly. Executor confirmed via LayerChart source.

## Fix 4 — CSS fallback
